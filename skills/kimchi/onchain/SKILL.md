# On-Chain Data Monitor

## Description
Track on-chain metrics: whale transactions, exchange flows, network health.

## Free Data Sources

### Whale Tracker
- Use web search for "whale alert bitcoin" 
- Check blockchain explorers via web
- Look for large tx warnings

### Exchange Flows
- CoinGlass: https://www.coinglass.com/zh/Tvl
- DefiLlama: https://defillama.com

### On-Chain Metrics
- Glassnode (some free)
- CryptoQuant (some free)
- LookIntoBitcoin (free charts)

## Key Metrics
1. **Exchange Reserves** - BTC leaving/exiting exchanges
2. **Whale Transactions** - Large transfers (>100 BTC)
3. **Network Hash Rate** - Mining health
4. **Active Addresses** - Network usage
5. **MVRV** - Unrealized profit/loss

## Usage
1. Search for recent whale activity
2. Check exchange flow data
3. Report on key metrics

## Example
```
!whale btc
!onchain eth
!flows btc
```
