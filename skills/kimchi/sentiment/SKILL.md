# Crypto Twitter Sentiment

## Description
Monitor crypto Twitter/X for sentiment, trending topics, and market-moving discussions.

## Usage
1. Use web_search to find recent tweets/news about:
   - Bitcoin ETF flows
   - Institutional adoption
   - Regulatory news
   - Whale movements
   - Technical analysis shared

2. Search queries:
   - "Bitcoin BTC whale movement on-chain"
   - "crypto ETF inflow outflow this week"
   - "BTC resistance support breakdown"
   - "crypto sentiment fear greed index"

## Sentiment Indicators
- Fear & Greed Index (alternative.me)
- Funding rates (checking via exchange APIs)
- Social volume trends
- News sentiment

## Output
Brief summary of:
- Current sentiment (Bullish/Bearish/Neutral)
- Key themes discussion
- Any breaking news
- Whale/insider activity

## Example
```
!sentiment btc
!news eth
```
