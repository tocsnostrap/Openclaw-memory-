# Kimchi - Crypto Trading Skills

## Overview
Kimchi has access to the following crypto trading capabilities:

### Core Skills
1. **Price Monitor** (`price-monitor/`) - CoinGecko API for real-time prices
2. **TradingView** (`tradingview/`) - Chart analysis via browser
3. **MEXC** (`mexc/`) - Exchange integration for market data
4. **Sentiment** (`sentiment/`) - Twitter/news sentiment scanning
5. **On-Chain** (`onchain/`) - Whale tracking, exchange flows

## Quick Commands
- `!price btc eth sol` - Get prices
- `!chart btc 4h` - Open TradingView chart
- `!sentiment btc` - Check market sentiment
- `!whale btc` - Check whale activity
- `!mexc btc` - MEXC market data

## Data Sources
- CoinGecko (free API)
- MEXC (free API)
- TradingView (browser)
- Web search (news/sentiment)
- On-chain aggregators

## Note
All skills use free/public APIs. No private keys required for data retrieval.
